# Install-VagrantPlugin

## SYNOPSIS
Installs the given vagrant plugin

## SYNTAX

```
Install-VagrantPlugin [-Plugin] <String> [<CommonParameters>]
```

## DESCRIPTION
Installs the given vagrant plugin

## EXAMPLES

### EXAMPLE 1
```
Install-VagrantPlugin -Plugin vagrant-sandbox
```
